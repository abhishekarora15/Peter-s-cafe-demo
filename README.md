# Peter’s Cafe — Demo Website

This is a polished, responsive, mobile-first demo website for **Peter’s Cafe London**. It focuses on a high-end, minimal luxury aesthetic suitable for attracting professionals and tourists in the Aldgate area.

Features
- Full-screen hero with cinematic visuals
- Sticky header with smooth transition
- Menu, About, Reviews, Visit/Contact sections
- Interactive Google Maps embed
- Mobile-first responsive layout
- Accessible (skip link, ARIA labels, focus styles)
- SEO-ready meta tags and LocalBusiness structured data
- Subtle animations and grain texture

How to run locally
1. Open `index.html` in your browser (or serve with a static server):
   - e.g., `npx http-server` or `python -m http.server`

Performance & Production Notes
- External images reference Unsplash; consider self-hosting optimized images.
- Consider minifying CSS/JS and using HTTP/2 or a CDN for production.
- Add sitemap.xml and robots.txt when deploying.

Accessibility & SEO
- Ensure correct alt text for images and sufficient color contrast.
- Update Open Graph images and canonical URL before publishing.

Deploy
- Works well on GitHub Pages, Netlify, Vercel or any static host.
